About Disaster Management Project

This project is focused on helping identify and respond to disasters by using social media data. It uses a combination of image classification and text classification models to determine whether a disaster is genuine based on tweets and their images.

The application also has a live scraper that fetches real-time tweets related to disasters and performs predictions on them to identify genuine cases.

The technologies used include Flask, MongoDB, Python, machine learning models, and Twitter's scraping mechanisms.
